# 🎬 MovieRec Sys - 影视推荐系统项目总结

## 一、项目概述

这是一个基于 **Python 3.11** 开发的智能影视推荐系统，融合了**大数据计算**、**自然语言处理**和**协同过滤**技术，为用户提供个性化电影推荐服务，同时提供强大的数据可视化后台。

## 二、技术架构

### 前端技术栈
- **NiceGUI**：基于 FastAPI + Vue 的现代全栈框架
- **Plotly**：动态数据可视化图表

### 后端技术栈
- **FastAPI**：高性能异步 Web 框架
- **SQLAlchemy**：ORM 数据库操作
- **PostgreSQL**：关系型数据库（支持异步 asyncpg 和同步 psycopg2）

### 大数据与AI
- **Apache Spark (PySpark)**：离线大规模推荐计算
- **PyTorch + Transformers**：深度学习 NLP 模型
- **Scikit-learn**：实时协同过滤算法
- **Pandas**：数据分析与处理

## 三、核心创新点

### 1. 混合推荐引擎（四大策略）

**离线推荐（Spark ALS）**
- 使用 PySpark 的交替最小二乘法处理海量评分数据
- 解决数据稀疏性问题，挖掘潜在 User-Item 关联
- 在 [spark_runner.py](cci:7://file:///d:/code/python/movie/spark_runner.py:0:0-0:0) 中实现，支持 RMSE 评估指标

**实时推荐（Item-Based CF）**
- 基于 Scikit-learn 余弦相似度计算
- 根据用户实时收藏和评分行为，毫秒级响应
- 在 [recommendation_service.py](cci:7://file:///d:/code/python/movie/services/recommendation_service.py:0:0-0:0) 中实现，支持模型持久化

**情感语义推荐（NLP）**
- 集成 Hugging Face 的 `chinese_xlm_xnli` 模型
- 支持自然语言输入（如"今天加班好累"）
- AI 自动识别情绪并推荐治愈系影片
- 在 [analysis_service.py](cci:7://file:///d:/code/python/movie/services/analysis_service.py:0:0-0:0) 中实现，包含7种情绪场景配置

**冷启动策略**
- 针对新用户提供基于热门榜单、分类精选的兜底推荐
- 确保系统对任何用户都有推荐结果

### 2. 数据可视化大屏

在 [admin_analytics.py](cci:7://file:///d:/code/python/movie/pages/admin_analytics.py:0:0-0:0) 和 [analysis_service.py](cci:7://file:///d:/code/python/movie/services/analysis_service.py:0:0-0:0) 中实现：
- **票房与评分气泡图**：商业价值分析
- **题材占比玫瑰图**：类型偏好分析
- **评分分布箱型图**：评分深度分析
- **题材变迁热力图**：时空演变分析
- **质量与热度散点图**：口碑与热度关系
- **中西审美对比图**：豆瓣 vs IMDb 评分对比

### 3. 智能情感交互

在 [user_home.py](cci:7://file:///d:/code/python/movie/pages/user_home.py:0:0-0:0) 中实现：

- **心情标签**：7种预设情绪（开心、难过、愤怒、害怕、刺激、平静、烧脑）
