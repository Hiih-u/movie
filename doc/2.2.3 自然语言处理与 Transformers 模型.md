### 2.2.3 自然语言处理与 Transformers 模型

在传统的影视推荐系统中，用户往往只能通过点击明确的分类标签（如“喜剧”、“动作”）或搜索特定的影片名称来获取推荐结果。这种交互方式忽略了影视作品作为情感载体的核心属性，也难以满足用户在特定情绪状态下（如“今天加班好累，想看点解压的”）的模糊检索需求。为了打破这一局限，本系统引入了自然语言处理（Natural Language Processing, NLP）技术，并深度融合了基于 Transformers 架构的预训练大语言模型，创新性地构建了基于认知与情感感知的交互模块。

**1. Transformers 架构与 Hugging Face 生态**
自 2017 年 Google 提出 Transformers 架构以来，自然语言处理领域迎来了范式级别的跨越。Transformers 摒弃了传统 RNN/CNN 的时序依赖，完全基于自注意力机制（Self-Attention Mechanism），能够高效地进行并行计算并精准捕获长文本中的全局语义依赖关系。
Hugging Face 则是目前全球最大的开源自然语言处理模型库与社区，其提供的 `transformers` 库以极其简练的 API 封装了当前主流的预训练模型（如 BERT、GPT、RoBERTa 等）。在本课题中，系统利用 `transformers` 库的 `pipeline` 接口构建了高效的自然语言推理流水线，这使得推荐系统具备了工业级的自然语言理解能力。

**2. `chinese_xlm_xnli` 模型解析**
为了在中文语境下实现精准的语义理解与情绪捕获，本系统集成了由 Hugging Face 社区开源的 `chinese_xlm_xnli` 预训练模型。
该模型是基于跨语言语言模型（Cross-lingual Language Model, XLM）架构，并在 XNLI（Cross-lingual Natural Language Inference，跨语言自然语言推理）中文语料库上进行了深度微调（Fine-tuning）的版本。自然语言推理（NLI）任务的核心在于判断给定的两个句子（前提 Premise 与假设 Hypothesis）之间是否存在蕴含（Entailment）、矛盾（Contradiction）或中立（Neutral）的关系。由于该模型在中文自然语言推理任务上表现出了极高的鲁棒性与语义对齐能力，它成为了构建后续零样本分类任务的理想基座模型。

**3. 零样本分类技术（Zero-Shot Classification）**
在实际的软件工程场景中，如果采用传统的监督学习方法来构建情感分类器，需要耗费巨大的人力成本去标注数十万条“用户输入的口语化短文本”与“离散情绪类别”的映射数据。为了规避这一成本，本系统采用了先进的**零样本分类（Zero-Shot Classification）技术**。
零样本分类是一种巧妙地利用自然语言推理（NLI）模型能力的迁移学习策略。其基本原理是：

* 将用户输入的自然语言文本（如“我今天失恋了，心情很低落”）作为 NLI 模型中的“前提（Premise）”。
* 将预定义的情感类别标签（如“难过”）构造为一个陈述句式（如“这句话表达了难过的情绪”）作为 NLI 模型中的“假设（Hypothesis）”。
* 模型通过计算“前提”蕴含“假设”的概率，即可得出该文本属于各个情感标签的置信度得分（Score）。

**4. 零样本分类在系统中的工程应用**
在 MovieRec Sys 的具体实现中，NLP 模型主要驱动前台的“情感树洞”模块。

* **服务层封装**：在 `services/analysis_service.py` 模块中，系统预定义了涵盖人类主要情绪的场景字典 `MOOD_SCENARIOS`，包括“开心”、“难过”、“愤怒”、“害怕”、“刺激”、“平静”和“烧脑”等候选标签。
* **模型推理流水线**：系统通过 `pipeline("zero-shot-classification")` 接口将本地的 `chinese_xlm_xnli` 模型加载至内存中进行单例缓存。
* **意图识别与过滤**：当接收到用户的文本输入后，`analyze_text_mood` 函数会将文本与预设的候选标签列表一同送入模型进行推理，并获取概率分布（Scores）。为了保证推荐结果的准确性与宁缺毋滥的体验，系统设置了 `0.3` 的置信度阈值（Threshold），仅当最高匹配标签的得分大于该阈值时，系统才会采纳该情绪标签，否则将提示用户更换说法。

通过集成上述 NLP 与大模型技术，本推荐系统赋予了自身理解用户非结构化语义的能力，成功地将复杂的深度学习技术落地为直观、温暖的交互功能，显著提升了系统的智能化水平。
